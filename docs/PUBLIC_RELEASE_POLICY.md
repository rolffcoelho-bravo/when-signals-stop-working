# Public Release Policy

## Included

- frozen raw SOL and BTC OHLCV snapshot;
- download and validation manifests;
- complete processed feature matrix;
- exact chronological fold assignments;
- complete out-of-sample predictions;
- summary tables, verdicts, report, and SVG figures;
- model contract, research protocol, references, roadmap, and citation metadata;
- sanitized runtime package versions and SHA-256 checksums.

## Excluded

- API keys, account data, credentials, `.env` files, or private keys;
- virtual environments, caches, packaging metadata, and IDE files;
- absolute local filesystem paths and machine-specific user information;
- private correspondence and internal positioning notes;
- temporary logs and optional high-resolution raster duplicates.

## Interpretation discipline

The release reports all declared indicators and folds. It does not suppress negative folds, select only favorable horizons, or revise the frozen V1 specification after observing the result.
