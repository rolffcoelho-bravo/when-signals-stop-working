# Public Release Policy

## Release objective

Provide a complete and professionally governed replication package while excluding credentials, private machine information, and non-essential generated artifacts.

## Included public evidence

- frozen SOL and BTC OHLCV snapshot;
- download, provenance, and validation records;
- aligned market data and complete model feature matrix;
- exact fold boundaries and observation-level assignments;
- complete out-of-sample predictions;
- summary tables, operational determinations, report, and vector figures;
- model contract, research protocol, status governance, references, roadmap, and citation metadata;
- sanitized runtime versions and SHA-256 checksums.

## Excluded material

- API keys, credentials, account data, `.env` files, or private keys;
- virtual environments, caches, packaging metadata, and IDE configuration;
- absolute local filesystem paths and machine-specific user information;
- private correspondence, internal positioning notes, and temporary logs;
- optional high-resolution raster duplicates where vector evidence is available.

## Publication standard

The release reports every declared indicator and chronological fold. It does not suppress negative evidence, select only favourable horizons, or revise the frozen Version 1 specification after observing the result.

## Data-rights statement

The MIT License applies to ShockBridge-authored code and documentation. It does not transfer ownership of third-party market data. The frozen snapshot is included for transparent replication and remains subject to the source venue's applicable terms and availability.
