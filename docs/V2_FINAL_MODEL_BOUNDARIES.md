# Version 2 Final Model Boundaries

## Permitted conclusion

Version 2 supports the conclusion that stable incremental value was not
established for either confirmatory signal family under the frozen primary-case
protocol.

## Prohibited conclusions

The evidence does not support claims that:

- RSI or Bollinger universally never works;
- the Bollinger pipeline has an operational trading edge;
- a positive average return establishes economic value;
- the filtered stress state proves investor panic;
- the result generalizes across venues, quote currencies, or assets;
- live execution, capacity, financing, liquidation, or order-book effects have
  been validated.

## Model-risk status

- RSI: `NO_PIPELINE_ADMITTED`
- Bollinger: `NO_INCREMENTAL_EVIDENCE`
- Primary case: not established
- External replication: not triggered for establishment
- Monitoring readiness: not granted
- Operational deployment: not supported
- Revalidation: required before any different use claim

## V2.1 boundary

The V2.1 panic-conditioned programme is a new study of signal governance under
stress. It cannot revise Version 2. Its state definitions, hypotheses, external
datasets, and prospective validation period must be frozen independently.
