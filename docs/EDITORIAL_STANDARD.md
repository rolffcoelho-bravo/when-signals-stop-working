# Institutional Editorial Standard

Public repository writing follows five requirements:

1. **Evidence before promotion** - claims are stated only where supported by the published outputs.
2. **Determination language** - use assessment, evidence, contribution, status, scope, and governance terminology rather than conversational or promotional framing.
3. **Explicit boundaries** - identify venue, sample, target, cost, data, and implementation limits.
4. **Reproducibility** - connect every substantive result to a tracked table, figure, manifest, or method contract.
5. **Restraint** - remove private correspondence, internal positioning language, redundant explanation, and unsupported adjectives.

The repository title may retain the accessible phrase "When Signals Stop Working". All supporting documentation must use formal institutional language and distinguish clearly between establishment, deterioration, and suspension.
