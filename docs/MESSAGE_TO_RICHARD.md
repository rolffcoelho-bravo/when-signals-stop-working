Richard, no problem. Bollinger Bands are related to RSI because both are often used to identify apparent market extremes, but they measure different things. RSI summarizes recent gain-loss momentum, while Bollinger Bands measure where price sits relative to a volatility-adjusted moving range.

I have kept your original RSI question in the framework rather than discarding it. The analysis now produces two separate answers:

1. Did RSI ever establish incremental predictive value for SOL, in which market regimes, and has that contribution subsequently deteriorated?
2. Did Bollinger Bands establish incremental value under the same validation rules?

For each indicator, the model follows three stages:

- test the conventional threshold event;
- compare it with a non-indicator model using trend, volatility, volume, recent SOL returns, and Bitcoin conditions;
- estimate the current range, trend, or stress regime and monitor structural deterioration sequentially.

The important distinction is that an indicator cannot be said to have stopped working unless it first demonstrated genuine out-of-sample value. If it never beats the benchmark, the correct conclusion is that its predictive value was not established. If it previously added value but later crosses the structural-change and recent-performance gates, it can then be classified as suspended.

For the initial demonstration, I am using four-hour SOL candles, RSI-14 with 30/70 thresholds, and 20-period Bollinger Bands with two standard deviations. The next four-hour return is the fixed forecast horizon.
